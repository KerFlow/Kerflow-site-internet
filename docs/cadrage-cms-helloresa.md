# Cadrage CMS et HelloResa

Note de suivi pour le cadrage client.

## Connexions HelloResa

Elements a connecter a HelloResa :

- Planning des cours : oui
- Reservation des cours : oui
- Ateliers : oui
- Evenements : oui

Elements a ne pas connecter a HelloResa :

- Rendez-vous / soins : non
- Location d'espaces : non

## Liens attendus

- "Reserver un atelier" conduit a HelloResa.
- "Voir tous nos ateliers" conduit au planning.

## Notes de vocabulaire

- Les profils professeurs et les profils intervenant-es / praticien-nes peuvent etre regroupes dans une meme logique CMS de "profils", avec une categorie ou un type de profil.
- CTA signifie "call to action", donc bouton ou lien d'action : reserver, contacter, voir le planning, faire une demande, etc.

## Contenus editables sur la home

- Les avis de la section "08 - Avis" sont geres dans le CMS via "Accueil - contenus editables".
- Chaque avis contient : le prenom / nom affiche, le contexte court et le texte de l'avis.
- Les clientes peuvent ajouter de nouveaux avis depuis ce tableau ; ils s'affichent automatiquement sur la page d'accueil.
