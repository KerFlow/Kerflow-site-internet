01 - Principe

Une palette principale commune, puis des couleurs de service en accent.

Le site KerFlow garde la direction Galet / Pierre & Mer comme base: fonds, textes, navigation, footer, boutons principaux et grandes sections. Les couleurs Studio, Consult, Rituel et Ateliers servent a identifier les entrees, les cartes, les pastilles, les details graphiques et certains boutons secondaires. Sur la homepage, elles doivent rester ponctuelles pour conserver une lecture calme et coherente.

Structure

Utiliser le lin, le papier, le petrol et le granite pour construire la page. Ce sont les couleurs de fondation.

Identification

Utiliser les couleurs de service pour aider a reperer une offre, pas pour recolorer toute la page.

Hierarchie

Garder le petrol pour les textes de marque, les liens importants et les CTA principaux.

04 - Vert Studio

Studio utilise une famille de verts limitee et hierarchisee.

Couleur Usage recommande Usage a eviter
#d4dcc6 Fond de carte Studio, pastilles, zone douce, aplats legers. Texte principal ou CTA seul: contraste insuffisant.
#8aa074 Accent secondaire, filet, hover, pictogramme, petit detail graphique. Grand aplat dominant sur toute la homepage.
#4f6b3c Accent fort Studio, icones, titres courts, boutons secondaires sur fond clair. Texte long ou remplacement du petrol pour le mot KerFlow.

05 - Logo & nom

KerFlow reste toujours ecrit en petrole.

KerFlow Studio · Consult · Rituel · Ateliers
KerFlow Version sombre / noir & blanc

A respecter

  • Le mot KerFlow est toujours en #1c4244 en version couleur.
  • La base logo principale utilise #e8e6dd.
  • La version noir et blanc peut utiliser #f1ebe1.

A eviter

  • Ne pas mettre KerFlow dans les couleurs de service.
  • Ne pas multiplier les verts au-dela des trois verts Studio valides.
  • Ne pas utiliser toutes les couleurs service en grands aplats simultanes.
